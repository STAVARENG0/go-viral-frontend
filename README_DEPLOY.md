# Go Viral Frontend

Atualização profissional mantendo o design atual.

Alterações principais:
- Card 1 reorganizado: Novo seguidor, Direct, Comentário.
- Passo 2 agora é uma estratégia editável.
- Cada gatilho possui estratégia e opções próprias.
- Opções e respostas podem ser editadas na configuração avançada.
- Front não salva mais chave no localStorage.
- Login usa sessão segura por cookie HttpOnly no backend.
- API base agora vem por variável de ambiente.

Variável do frontend:

```env
VITE_API_BASE_URL=https://api.seudominio.com
```
